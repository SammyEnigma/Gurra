#IconFonts object holds all app icon fonts

##Properties

###**primary**: 

    default: fontAwesome
    
This property holds the default icon font name

###**afiado**: string
This property holds the name of loaded Afiado font
###**brankit**: string
This property holds the name of loaded Brankit font

###**feather**: string
